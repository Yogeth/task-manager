# Task Manager

A simple and responsive Task Manager application built with React and Vite. This application allows users to create, complete, update, and delete tasks while providing a clean and user-friendly interface.

## Features

* Add new tasks
* Mark tasks as completed
* Update existing tasks
* Delete tasks
* Responsive design
* Fast performance with Vite
* Simple and intuitive user interface

## Technologies Used

* React
* Vite
* JavaScript (ES6+)
* Tailwind CSS
* CSS3

## Installation

1. Clone the repository:

```bash
https://github.com/Yogeth/task-manager.git
```

2. Navigate to the project directory:

```bash
cd task-manager
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Usage

1. Enter a task in the input field.
2. Click the **Add** button to create a new task.
3. Click the check icon to mark a task as completed.
4. Click the delete icon to remove a task.
5. Manage your tasks efficiently from a single interface.
